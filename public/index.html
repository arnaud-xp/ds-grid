About Containers

This is a .container. Its purpose is to handle your HTML code and resize automatically with the viewport.

If you want your container to use full parent width just add .fluid class to it.

The grid

Basics

DS Grids are responsive and mobile first by using "min-width" media queries.

By default, DS Grid handles 4 breakpoints :

  • Xtra small viewports (xs): < 768px
  • Small viewports (sm): >= 768px
  • Medium viewports (md): >= 992px
  • Large viewports (lg): >= 1200px

Rows & cols

A grid is made of 2 base elements : .rows containing .cols.
